This is a modified front-end for BlinkFill derived from BlinkFill Demo made availale by Rishab Singh at Microsoft Research

    How to use BlinkFill Demo:
  1. Load Data: Load the input data by copy-pasting or uploading a csv file.
  2. Output Example: Provide the output examples next to the right-most column of the input data. You are OK to provide examples for any rows.
  3. Start Synthesis Process: Click "BlinkFill" button. If successfull, the system will fill the empty cells with synthesized data.
